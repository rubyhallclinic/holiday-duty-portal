Step 1 — Select Holiday & Download Submitted List

Download the submitted employee list to fill validation status in Excel

How this works:
1. Select a holiday below and download the submitted employee list as Excel
2. Open the Excel, verify each employee's IN/OUT attendance logs in your attendance system
3. Fill the Validation Status column and optionally add Remarks
4. Save and upload the filled Excel back in Step 3

Step 2 — Fill Validation Status in Excel

Open the downloaded file and update the Validation Status column

Excel Columns — Do Not Change Column Order

Emp Code PF No. Employee Name Department Location Holiday Name Holiday Date Validation Status ★ Remarks (optional)
  • Only the Validation Status and Remarks columns need to be filled — do not modify other columns
  • Do not delete or add rows — only fill the status against each employee
  • Re-upload allowed anytime — latest upload always replaces previous status

Suggested Validation Status Values

✓ APPROVED ✗ REJECTED — NO ATTENDANCE LOGS FOUND ✗ REJECTED — PARTIAL DUTY ✗ REJECTED — EARLY DEPARTURE ✗ REJECTED — LATE ARRIVAL ⧖ PENDING VERIFICATION
Click any status above to copy it. You can also type any custom status — the system accepts any text.
